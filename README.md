# MiCoreView

MiCoreView is a centralized infrastructure management platform for monitoring, organizing, and operating devices across pods, racks, and servers. Rather than managing servers and other infrastructure assets one by one, MiCoreView provides operators with a single web console to view dashboard gadgets for out-of-band and in-band metrics, device inventory management, location hierarchy views, group-based device organization, discovery workflows, alerts, events, control tasks, extension access, and system administration settings.

Users can use MiCoreView to discover and register devices, filter inventory by type, model, tenant, IP range, or physical location, and view detailed gadget panels for servers and other managed equipment. With built-in location hierarchy management across pods, areas, rows, racks, and devices, MiCoreView gives operators both logical and physical context for managed assets.

When operational action is needed, MiCoreView provides workflows for power control, firmware provisioning, OS provisioning, OS backup and restore, backup configuration, factory shipping SKU checks, and LED control. Alert and event pages give teams visibility into system activity by providing rule management, log review, severity/type filtering, notes. The Extension Hub introduces add-on workflows such as Kubespray-based Kubernetes deployment and other extension packages. Administrative screens cover account management, SMTP settings, audit logs, certificate upload, system preferences, refresh behavior, import/export settings, and notification display options.

MiCoreView is designed for infrastructure and operations teams that need a single place to monitor assets, organize device fleets, and carry out day-to-day management tasks across distributed environments.
