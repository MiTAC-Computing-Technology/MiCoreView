# MiCoreView

MiCoreView is a pod management user interface for monitoring, organizing, and operating infrastructure devices from a centralized web console. It provides dashboard gadgets for out-of-band and in-band metrics, device inventory management, location hierarchy views, group-based device organization, discovery workflows, alerts, events, control tasks, extension access, and system administration settings.

The UI is organized around common pod operations. Users can add and inspect devices, filter inventory by type, model, tenant, IP range, or physical location, and view detailed gadget panels for servers and other managed equipment. MiCoreView also supports location hierarchy management across data centers, areas, rows, racks, and devices, giving operators both logical and physical context for managed assets.

For operations teams, MiCoreView includes control workflows such as power control, firmware provisioning, OS provisioning, OS backup and restore, backup configuration, factory shipping SKU checks, and LED control. Alert and event pages provide rule management, log review, severity/type filtering, notes, and operational visibility. The Extension Hub introduces add-on workflows such as Kubespray-based Kubernetes deployment and other extension packages.

Administrative screens cover account management, SMTP settings, audit logs, certificate upload, system preferences, refresh behavior, import/export settings, and notification display options.
